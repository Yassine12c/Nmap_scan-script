# Nmap Scan Script

Script Bash permettant de scanner une IP ou un site web avec l'outil **Nmap**.

## Fonctionnalités
- Scan rapide
- Scan complet
- Détection des services
- Scan avancé (OS + scripts)
- Ping scan (machine active)

## Usage
1. Rendre le script exécutable :

```bash
chmod +x scan.sh

Lancer le script :

./scan.sh

Suivre les instructions affichées dans le terminal.

Avertissement

⚠️ Ce script est destiné uniquement à des tests éducatifs et légaux.
Ne scannez jamais de sites ou d’IP sans autorisation.
Utilisez ce script uniquement dans votre labo ou sur des machines de test comme scanme.nmap.org.
